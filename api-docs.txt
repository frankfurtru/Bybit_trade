# Bybit V5 API - Semua Endpoint Documentation

Bybit V5 API menggunakan struktur penamaan yang konsisten dengan format: `{host}/{version}/{module}` dimana semua endpoint menggunakan path `/v5/`. API ini menyatukan trading untuk Spot, Derivatives, dan Options dalam satu set spesifikasi.[1]

## Struktur API Endpoints

### 1. Market Data Endpoints (`/v5/market/`)
Endpoint untuk data pasar real-time dan historis - tidak memerlukan autentikasi:[2][1]

- **GET /v5/market/time** - Get Bybit Server Time[2]
- **GET /v5/market/kline** - Get kline/candlestick data[2]
- **GET /v5/market/mark-price-kline** - Get Mark Price Kline data[2]
- **GET /v5/market/index-price-kline** - Get Index Price Kline data[2]
- **GET /v5/market/premium-index-price-kline** - Get Premium Index Price Kline data[2]
- **GET /v5/market/instruments-info** - Get launched instruments information[3][2]
- **GET /v5/market/orderbook** - Get order book data[3][2]
- **GET /v5/market/tickers** - Get all latest information of symbols[3][2]
- **GET /v5/market/funding-history** - Get historical funding rate[2]
- **GET /v5/market/recent-trade** - Get public trade history[3][2]
- **GET /v5/market/open-interest** - Get open interest[2]
- **GET /v5/market/historical-volatility** - Get historical volatility[2]
- **GET /v5/market/insurance** - Get Insurance fund data[2]
- **GET /v5/market/risk-limit** - Get risk limit[2]
- **GET /v5/market/delivery-price** - Get delivery price[2]
- **GET /v5/market/long-short-ratio** - Get long short ratio[2]

### 2. Order Management Endpoints (`/v5/order/`)
Endpoint untuk manajemen order trading:[1][3]

- **POST /v5/order/create** - Place new orders untuk Spot, Margin, USDT perpetual, USDT futures, USDC perpetual, USDC futures, Inverse Futures dan Options[4][3]
- **POST /v5/order/amend** - Modify existing orders[3]
- **POST /v5/order/cancel** - Cancel active orders[3]
- **POST /v5/order/create-batch** - Place multiple orders[5]
- **GET /v5/order/history** - Query historical orders[3]
- **GET /v5/order/realtime** - Get active orders
- **POST /v5/order/cancel-all** - Cancel all orders

### 3. Position Management Endpoints (`/v5/position/`)
Endpoint untuk manajemen posisi trading:[6][1]

- **GET /v5/position/list** - Get Position Info[6][3]
- **POST /v5/position/set-leverage** - Set leverage[6][3]
- **POST /v5/position/switch-isolated** - Switch Cross/Isolated Margin[6]
- **POST /v5/position/set-tpsl-mode** - Set TP/SL Mode[6]
- **POST /v5/position/switch-mode** - Switch Position Mode[6]
- **POST /v5/position/set-risk-limit** - Set risk limit[6]
- **POST /v5/position/trading-stop** - Set trading stop (take profit/stop loss)[3][6]
- **POST /v5/position/set-auto-add-margin** - Set Auto Add Margin[6]
- **POST /v5/position/add-margin** - Add Or Reduce Margin[6]
- **GET /v5/position/execution-list** - Get Execution list (2 years)[6]
- **GET /v5/position/closed-pnl** - Get Closed PnL[6]

### 4. Account Management Endpoints (`/v5/account/`)
Endpoint untuk operasi akun tunggal:[7][1]

- **GET /v5/account/wallet-balance** - Get wallet balance[7][3]
- **POST /v5/account/upgrade** - Upgrade to Unified Account[7]
- **POST /v5/account/set-collateral-coin** - Set collateral coin[7]
- **GET /v5/account/borrow-history** - Get borrow history[7]
- **GET /v5/account/collateral-info** - Get collateral info[7]

- **POST /v5/account/set-margin-mode** - Set margin mode[7]
- **POST /v5/account/set-spot-hedging** - Set spot hedging for Portfolio margin[7]

### 5. Asset Management Endpoints (`/v5/asset/`)
Endpoint untuk operasi lintas akun - manajemen aset dan dana:[9][1]

## Fitur Utama API V5

### Authentication
- **Public Endpoints**: Market data tidak memerlukan API key[8]
- **Private Endpoints**: Memerlukan API key dan secret untuk autentikasi[8]

### WebSocket Endpoints
- **Public Stream**: `wss://stream.bybit.com/v5/public/linear` (untuk linear)
- **Public Stream**: `wss://stream.bybit.com/v5/public/spot` (untuk spot)
- **Private Stream**: `wss://stream.bybit.com/v5/private`[10]

### Rate Limits
API V5 memiliki batasan rate limit yang berbeda untuk setiap endpoint. Untuk menaikkan rate limit, hubungi client manager atau submit aplikasi.[4]

### Order Limits
- **Perps & Futures**: Maksimal 500 active orders per symbol, 10 conditional orders per symbol[4]
- **Spot**: 500 orders total, termasuk maksimal 30 TP/SL orders dan 30 conditional orders per symbol[4]
- **Options**: Maksimal 50 open orders per akun[4]

API V5 mendukung berbagai kategori produk yang dapat dispecify melalui parameter `category`: `spot`, `linear`, `inverse`, dan `option`.[1][4]

# Bybit V5 API - Dokumentasi Lengkap dengan Request Examples

Bybit V5 API menyatukan seluruh fitur trading untuk Spot, Derivatives (Linear/Inverse), dan Options dalam satu set spesifikasi yang konsisten. API V5 mendukung Unified Trading Account (UTA) dan Classic Account dengan struktur URL: `{host}/{version}/{module}`.[1]

## Base Configuration

**Production**: `https://api.bybit.com`

### Struktur Address Segment[1]
- **v5/market/**: Candlestick, orderbook, ticker, data transaksi platform, aturan finansial, aturan kontrol risiko
- **v5/order/**: Manajemen order
- **v5/position/**: Manajemen posisi  
- **v5/account/**: Operasi akun tunggal - akun funding terpadu, rates, dll
- **v5/asset/**: Operasi lintas akun - manajemen aset, manajemen dana, dll
- **v5/spot-lever-token/**: Quote dari Leveraged Tokens di Spot, fungsi beli dan tebus
- **v5/spot-margin-trade/**: Manajemen Margin Trading di Spot

### Unified Account Features[1]
- **Capital Efficiency**: Sharing dana lintas Spot, USDT Perpetual, USDC Perpetual dan Options
- **Cross-utilization**: Offset profit dan loss lintas posisi berbeda
- **Borrowing**: Pledge multiple assets sebagai collateral untuk margin trading
- **Portfolio Margin**: Combined margin antara berbagai kontrak

### Category Parameter[5][1]
- `spot`: Spot trading
- `linear`: USDT/USDC perpetual/futures
- `inverse`: Inverse perpetual/futures  
- `option`: Options trading

### Rate Limits & Order Limits[5]
- **Perps & Futures**: Max 500 active orders per symbol
- **Spot**: Max 500 orders total, max 30 TP/SL orders per symbol
- **Options**: Max 50 open orders per account
- **Rate Limits**: Bervariasi per endpoint, gunakan SDK untuk limit lebih tinggi[8]

API V5 menyediakan uniformitas dan efisiensi untuk seluruh lini produk Bybit, menyatukan Spot, Derivatives, dan Options dalam satu set spesifikasi dengan global dictionary yang konsisten.
